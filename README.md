# Magic 8 Ball Zulip Bot

This is repo for a Recurse Center [zulip](https://en.wikipedia.org/wiki/Zulip) bot that will interact in Zulip discussion forums when called upon. The bot will respond with one of the traditional magic 8 ball responses.

Tech Notes:  
-To call the bot, tag @magic8-bot in your Zulip post

### running locally for testing

```
uv run python magic8.py
and include ./zuliprc (the Zulip config file)
```
